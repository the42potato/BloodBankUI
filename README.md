# BloodBankUI
A UI designed for integration with a blood bank's collection and donation database

## Dependencies
- Java 17
- Maven 3.10.1
- JavaFX 17.0.2
- MySQL 8.0.33

## Installation
Set up locally with these steps:
1. Clone the repository
`git clone https://github.com/the42potato/BloodBankUI.git`
2. Navigate to the project directory
3. Install dependencies
`npm install`
